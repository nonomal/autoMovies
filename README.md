# 我只想好好观影
本脚本的目的是为了最小化观影的门槛。

使用说明：
安装完毕后，打开豆瓣，搜索自己想看的电影，点击播放。结束。

# 本脚本特色
1. 极度简洁
2. 自动测速
3. 超级快速
4. 无畏并发，因为它根本没有后端服务器哈哈哈，理论ddos承载量为无限，哈哈哈哈哈。

# 本脚本影片来源
你不用管

# 本脚本缺点
影片不会像百度网盘或者阿里网盘那么清晰且可能有采集站本身的广告，但是胜在快！不用找资源，一键直达！

# 开发的背后
我是尝试了许多让我爸妈看电影的方式，NAS，jellyfin，AList，无一例外，都很复杂，我只想让他们能轻松的看电影和看电视剧。
